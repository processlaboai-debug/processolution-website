# ProcesSolution — Istruzioni per Claude Code

## Lingue
Il sito è disponibile in tre lingue: **inglese (EN)**, **italiano (IT)** e **lituano (LT)**.

Per tutte le lingue, usa i rispettivi dizionari linguistici per assicurare l'uso di parole ed espressioni corrette, naturali e appropriate al contesto professionale:
- **EN** — inglese standard internazionale, tono professionale e diretto
- **IT** — italiano corretto, evita calchi dall'inglese dove esiste una forma italiana naturale
- **LT** — lituano corretto, usa le virgolette tipografiche „ " (U+201E / U+201D), non le virgolette ASCII `"`
